# AndroidPanel
安卓性能监控面板，旨在将性能测试自动化、规范化

1.运行环境
------------
- python3+
- adb & devices

1.1虚拟环境(推荐)
- pip install pipenv
- pip shell
- pip install

1.2本机环境
- pip install -r requirements.txt

2.运行
------------
- flask forge（初始化数据库）
- flask run
- 浏览器打开 http://127.0.0.1:5000/
